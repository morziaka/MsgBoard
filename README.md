Проект "Доска объявлений".
Реализована регистрация пользователя, возможность добавить объявление для зарегистрированного пользователя и оставить отклик к объявлениям на портале. 
В объявлении добавлено поле с возможностью вставки фото, файла или видео с помощью библиотеки Froala Editor.
Автору объявление приходит уведомление о новом отклике на email. Автор может принять либо отклонить объявление.
Есть страница с категориями, где авторизованный пользователь может подписаться на одну или несколько категорий.
Также есть еженедельная рассылка подписчикам категорий с новыми постами в этих категориях.
